"Você tem em mãos o projeto do Figma do Nossa Doçura, uma empresa pequena com um curto catálogo de doces.

A empresa quer criar um projeto de Onepage apenas para divulgá-los e mostrar o que oferecem aos seus clientes.

Nessa atividade você terá 3 etapas de conclusão:
Finalização do protótipo de alta fidelidade com aplicação da identidade visual da marca
Marcação em html das seções header, intro, clássicos e footer
Estilização das seções descritas acima

O que será avaliado:
Manipulação de elementos utilizando Figma
Aplicação da ID visual seguindo os princípios do design
Organização de pasta de projeto
Identificação dos elementos e marcação adequada em tags HTML
Marcação semântica em HTML
Estilização seguindo o modelo final definido no Figma (na etapa 1)"
